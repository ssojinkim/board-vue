Thank you for purchasing Minible Vue Nuxt JS Admin & Dashboard Template.

Minible is a simple and beautiful admin dashboard template built on top of awesome Bootstrap 5 and Vue Nuxt JS.

You can build any type of web application like Saas based interface, eCommerce, CRM, CMS, Project management apps, Admin Panels, etc of any project.

Minible Vue Nuxt JS Admin & Dashboard Template is easily customizable and developer-friendly codes. It will help your team moving faster and saving development costs and valuable time.
